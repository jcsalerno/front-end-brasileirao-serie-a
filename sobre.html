Sobre a API Brasileirão Série A 2025

A API que fornece dados sobre times e jogadores do Campeonato Brasileiro Série A.

Sobre o Projeto

A API Brasileirão Série A 2025 foi desenvolvida para fornecer informações detalhadas sobre os times e jogadores que participam do Campeonato Brasileiro Série A. Com ela, você pode acessar dados como posição dos jogadores, times, e muito mais, de forma fácil e organizada.

Esta API é ideal para desenvolvedores que desejam integrar dados esportivos em suas aplicações, sejam sites, apps ou sistemas internos.

Nota: Este projeto é apenas para fins educativos e não tem fins lucrativos.

Funcionalidades

  • Consultar jogadores por posição (ex: Goleiro, Atacante).
  • Listar todos os times participantes do Brasileirão Série A 2025.
  • Buscar jogadores de um time específico.
  • Filtrar jogadores por time e posição.
  • Consultar informações detalhadas sobre jogadores, como nome, número, posição, data de nascimento, cidade, foto e nacionalidade.
  • Consultar informações sobre times, como nome, cidade sede, estádio e ano de fundação.

Tecnologias Utilizadas

Spring Boot

Framework Java para desenvolvimento de APIs RESTful.

PostgreSQL

Banco de dados relacional utilizado para armazenar os dados da API.

Render

Plataforma de hospedagem onde a API e o banco de dados estão implantados.

Bootstrap

Framework CSS para design responsivo e moderno.
